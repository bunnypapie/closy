Closy - เว็บไซต์จัดชุดเสื้อผ้า

วิธีเปิดใช้งาน
1) เปิดไฟล์ index.html ในเบราว์เซอร์
2) หรือใช้ Live Server / local server เพื่อใช้งานสะดวกขึ้น

หน้าที่มีให้
- index.html       หน้าแรก
- upload.html      เพิ่มเสื้อผ้า
- wardrobe.html    ดู/แก้ไข/ลบเสื้อผ้า
- generate.html    สุ่มและแนะนำชุด
- saved.html       ชุดที่บันทึก
- login.html       เข้าสู่ระบบและซิงก์ข้อมูล

ระบบที่ทำงานได้
- อัปโหลดรูปจากเครื่องและ preview
- บันทึกเสื้อผ้าลง localStorage
- เลือกหมวด Mood Occasion Weather
- แก้ไข/ลบเสื้อผ้า
- สุ่มชุดจากข้อมูลจริงใน wardrobe
- บันทึกชุดที่ชอบ
- ระบบ login แบบเดโม และ sync cloud จำลองผ่าน localStorage

หมายเหตุ
- โครงงานนี้เป็นเว็บฝั่ง Frontend พร้อมใช้งานสำหรับเดโมและส่งงาน
- หากต้องการใช้ cloud จริงในอนาคต สามารถต่อ Firebase / Supabase ได้
